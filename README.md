# WebAngularForXiuxiu - 医案管理 H5

## init(node: version > 12, cnpm)
    cnpm i

## run 
    npm run start-o

## build 
    npm run build-prod

## 配置文件 src/environments/*

    environment.api 当前使用的后台接口地址

    npm run start-o 使用的 environment.ts
    npm run build-prod 使用的 environment.prod.ts
